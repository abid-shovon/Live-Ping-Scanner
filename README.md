# Live-Ping-Scanner
###  Features

 **Auto OS Detection** (Windows/Linux)
 **Ping a Single IP Address**
 **Scan a Custom IP Range**
 **Cross-platform Support**
 **Uses Native Ping Commands**
 **ICMP TTL-based Live Host Detection**
 **Beginner-Friendly Code Structure**

###  Requirements

- Python 3.12
- Works on:
  - Windows (CMD)
  - Linux (Bash, Terminal)
